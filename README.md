## About the project
This simple project on Python allows you to use linux-like commands in `cmd.exe` on Windows.
I made:
```sh
ls
touch
rm
rm -r
cat
pwd
mv
cp
clear
```

## Installation
1. Run `pip install -r requirements.txt`
2. Run `py _setup.py` with administrator rights
